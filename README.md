# Lumi AI Beauty Commerce Platform

A modern AI-powered beauty e-commerce platform built with Next.js.

## Features

- Product browsing
- Product detail pages
- Shopping cart system
- User authentication
- Admin dashboard
- Responsive design
- Modern UI/UX

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Stripe
